# Crimson Compass Visual Assets Production Plan

## Overview
Complete visual asset creation plan for Crimson Compass iPhone mystery game. All assets will use AI generation with the prompts in `Assets/Art_Prompts/`. Target: Professional mobile game quality, mystery atmosphere, consistent Victorian/steampunk style.

## Phase 1: Core UI Elements (Week 1-2)
**Priority: HIGH** - Essential for basic gameplay

### Step 1.1: Main Menu UI (2-3 days)
- **Title Logo**: Crimson Compass text with ornate Victorian styling
- **Menu Buttons**: Play, Settings, Credits (3 variants: normal/hover/pressed)
- **Background Panel**: Subtle mystery-themed backdrop
- **Tools**: Midjourney, --ar 16:9
- **Quality Check**: High contrast, readable on mobile

### Step 1.2: Game UI Components (3-4 days)
- **Dialogue Box**: Ornate frame with parchment texture
- **Inventory Slots**: Antique wooden frames (5-10 variations)
- **Evidence Icons**: Magnifying glass, notebook, clue markers
- **Navigation Buttons**: Back, forward, menu (with hover states)
- **Tools**: DALL-E for clean vectors, Midjourney for detailed
- **Quality Check**: Touch-friendly sizing (44pt minimum)

### Step 1.3: HUD Elements (2-3 days)
- **Health/Time Indicators**: Victorian clock faces, gauge styles
- **Notification Bubbles**: Clue found, time running out
- **Progress Bars**: Investigation completion meters
- **Tools**: Consistent with UI style
- **Quality Check**: Clear visual hierarchy

## Phase 2: Backgrounds & Environments (Week 3-5)
**Priority: HIGH** - Sets the mystery atmosphere

### Step 2.1: Core Investigation Scenes (4-5 days)
- **Victorian Library**: Bookshelves, hidden passages, dust motes
- **Foggy Alley**: Crime scene, gas lamps, cobblestones
- **Mansion Interior**: Hallways, rooms, atmospheric lighting
- **Underground Chamber**: Ancient stone, arcane symbols
- **Rooftop Scene**: Urban night, chimney pots, city lights
- **Tools**: Midjourney --ar 9:16, focus on atmospheric depth
- **Quality Check**: Immersive composition, mobile performance

### Step 2.2: Transition & Loading Screens (2-3 days)
- **Loading Backgrounds**: Mystery-themed placeholders
- **Scene Transition Effects**: Fog, page turns, clock hands
- **Menu Backgrounds**: Consistent with game world
- **Tools**: Simplified versions of main backgrounds
- **Quality Check**: Fast loading, seamless transitions

### Step 2.3: Environmental Details (3-4 days)
- **Interactive Elements**: Doors, windows, furniture highlights
- **Ambient Details**: Cobwebs, dust, weather effects
- **Lighting Variations**: Day/night, interior/exterior
- **Tools**: Detail-focused prompts
- **Quality Check**: Enhances immersion without clutter

## Phase 3: Characters & NPCs (Week 6-8)
**Priority: MEDIUM** - Brings story to life

### Step 3.1: Main Characters (3-4 days)
- **Protagonist Detective**: Multiple expressions (determined, puzzled, triumphant)
- **Supporting Characters**: Partner, mentor, rival
- **Portrait Variations**: Different lighting, angles
- **Tools**: Midjourney --ar 3:4, focus on expressions
- **Quality Check**: Emotional readability, style consistency

### Step 3.2: NPCs & Suspects (4-5 days)
- **Victorian NPCs**: Butler, maid, shopkeeper, police officer
- **Suspects**: Various ages/genders, suspicious expressions
- **Witnesses**: Frightened, nervous, helpful expressions
- **Tools**: Character-focused prompts with personality traits
- **Quality Check**: Distinctive features, period-appropriate clothing

### Step 3.3: Character Integration (2-3 days)
- **Silhouettes**: For dramatic scenes
- **Partial Views**: Hands, shadows, mysterious figures
- **Animation Frames**: Basic gesture cycles if needed
- **Tools**: Simplified character prompts
- **Quality Check**: Consistent with full portraits

## Phase 4: Props, Evidence & Collectibles (Week 9-10)
**Priority: MEDIUM** - Essential for gameplay

### Step 4.1: Evidence Items (3-4 days)
- **Documents**: Letters, diaries, contracts with aged paper
- **Artifacts**: Keys, compasses, mysterious objects
- **Weapons**: Daggers, poisons, period-appropriate tools
- **Tools**: Detailed material textures, atmospheric presentation
- **Quality Check**: Clear identification, mystery appeal

### Step 4.2: Interactive Props (3-4 days)
- **Furniture**: Desks, cabinets, safes with hidden compartments
- **Decorations**: Paintings, clocks, mysterious devices
- **Clues**: Hidden messages, symbols, puzzle pieces
- **Tools**: Functional design with decorative elements
- **Quality Check**: Interactive potential, environmental integration

### Step 4.3: Collectible Items (2-3 days)
- **Achievement Badges**: Investigation milestones
- **Case Files**: Completed mystery summaries
- **Special Items**: Rare finds, easter eggs
- **Tools**: Consistent with evidence style
- **Quality Check**: Reward feeling, collection value

## Phase 5: Polish & Special Effects (Week 11-12)
**Priority: LOW** - Enhances experience

### Step 5.1: Visual Effects (3-4 days)
- **Particles**: Dust, fog, magical effects
- **Lighting Effects**: Glows, shadows, lens flares
- **Transition Effects**: Page turns, reveals, discoveries
- **Tools**: Effect-focused prompts
- **Quality Check**: Performance impact, visual impact

### Step 5.2: UI Polish (2-3 days)
- **Animations**: Button presses, panel slides
- **Feedback Effects**: Clue found, error states
- **Loading Animations**: Mystery-themed progress
- **Tools**: Motion-focused design
- **Quality Check**: Smooth, responsive feel

### Step 5.3: Final Optimization (2-3 days)
- **Resolution Check**: All assets at 1080p equivalent
- **Compression**: File size optimization for mobile
- **Consistency Review**: Style and quality across all assets
- **Tools**: Batch processing, quality assurance
- **Quality Check**: Professional game standard

## Tools & Resources

### Primary Tools
- **Midjourney**: Main generation tool for detailed scenes
- **DALL-E**: UI elements, clean graphics
- **Stable Diffusion**: Free alternative, custom models
- **Photoshop/GIMP**: Post-processing, cleanup

### Organization
- **Asset Folders**: UI/, Backgrounds/, Characters/, Props/
- **Naming Convention**: asset_type_variant_state.png
- **Version Control**: Git LFS for large image files

### Quality Assurance
- **Mobile Testing**: iPhone preview at actual size
- **Performance**: File sizes under 1MB per asset
- **Consistency**: Shared style guide reference
- **Feedback Loop**: Regular review and iteration

## Timeline & Milestones

- **Month 1**: Core UI complete, basic backgrounds
- **Month 2**: All environments, main characters
- **Month 3**: Props, evidence, polish effects
- **Total**: 12 weeks for complete visual suite

## Risk Mitigation
- **Backup Tools**: Multiple AI services for reliability
- **Iterative Process**: Generate, review, refine cycle
- **Fallback Assets**: Simple placeholders if needed
- **Scope Management**: Focus on essential assets first

## Success Criteria
- All core gameplay visuals complete
- Consistent mystery/Victorian aesthetic
- Mobile-optimized performance
- Professional quality suitable for App Store
