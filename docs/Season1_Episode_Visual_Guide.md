# Crimson Compass Season 1 Visual Generation Guide
# Episode-by-Episode Asset Creation with Grok Imagine

## Overview
This guide provides detailed instructions for generating all visual assets for Season 1 Episodes 1-4 using Grok Imagine. Each episode includes backgrounds, characters, objects, UI elements, and dynamic scenes.

**Generation Workflow:**
1. Visit [grok.com/imagine](https://grok.com/imagine)
2. Copy the prompt for each asset
3. Generate 3-5 variations per asset
4. Save with naming convention: `{Episode}_{AssetName}_{Variation}_{Date}.png`

### Batch Generation Commands:
```powershell
# Generate all Episode 1 assets
.\scripts\batch_generate_grok_assets.ps1 -Filter "S01E01_*"

# Generate all backgrounds
.\scripts\batch_generate_grok_assets.ps1 -Filter "*Background*"

# Generate all characters
.\scripts\batch_generate_grok_assets.ps1 -Filter "*Character*"
```
