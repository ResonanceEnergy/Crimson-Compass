## Scene 1: Access Control Investigation (2 minutes)

**Background:** Corporate headquarters with security checkpoints, badge readers, and access corridors. Badge cloning creating interference patterns. Service elevators and maintenance tunnels for stealth approaches.

**Narrative:** The team investigates a badge cloning operation targeting high-security access points. The Crimson Compass detects cloning signatures, with Gasket's memories surfacing about similar access breaches.

**Compass Mini-Puzzle:** Cloning Signature Analysis - Anomaly Detection
- **Objective:** Use compass to detect badge cloning patterns in access logs
- **Mechanics:** Scan access systems for cloning signatures, identify anomaly clusters, trace badge replication patterns
- **Solution:** Cloning signatures reveal master badge location and access breach routes
- **Reward:** Access to hidden cloning equipment and access log anomalies

**Treasure Hunting Element:** Headquarters Exploration - Relic Collection
- **Choice:** Direct security checkpoint access vs. maintenance tunnel infiltration
- **Consequence:** Checkpoint access yields access relics but risks triggering alerts; maintenance tunnel reveals cloning equipment evidence

**Dialogue:**

**ZTech:** "Their security is laughable! Check this badge scanner I built - it also brews coffee and predicts when it'll rain! Maximum absurdity, maximum function!"

**Narrative:** The confrontation reveals the IT specialist was running the cloning operation. The Crimson Compass detects access log anomalies that suggest a coordinated infiltration network.
