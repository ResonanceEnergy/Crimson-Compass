# S01E09: Exit Wound

## Episode Overview
**Title:** Exit Wound
**Case:** Physical extraction attempt of a prototype chip
**Case Token:** Extraction stopped; chip secured
**Shadow Token:** Exit hardware standardized across unrelated sites
**Theme:** Physical security, extraction operations

- **Surveillance Cameras** (OBSERVE): Monitor intruder movement

2. **Breach Detection:** Identify and track extraction team movement
