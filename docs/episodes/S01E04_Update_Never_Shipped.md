# S01E04: The Update That Never Shipped

## Episode Overview
**Title:** The Update That Never Shipped
**Case:** Firmware staging server hijacked
**Case Token:** Server reclaimed; breach sealed
**Shadow Token:** Deployment timestamps show the same skew
**Theme:** Cyber security, digital forensics

## Adventure Game Structure

### Starting Location: IT Department Lobby
**Description:** Corporate IT department entrance with reception desk, security checkpoints, and employee workstations.

**Interactive Objects:**
- **Visitor Log Terminal** (ENGAGE): Sign in as consultant
- **Security Badge Reader** (PROTOCOL): Access verification
- **Network Status Display** (OBSERVE): Current system status
- **Server Room Door** (KIT): Requires special access

**Initial Inventory:**
- IT Consultant Badge
- Network Diagnostic Tool
- Encrypted USB Drive
- Access Override Device

### Key Characters (Text-Based Dialogue)

#### Helix (Administrative Coordinator)
**Description:** Coordinating with IT management and providing operational support.
**Dialogue Tree:**
- Access coordination: "I've got you cleared as a security consultant. Play the part."
- System status: "Network logs show unusual traffic. Something's not right."
- Resource allocation: "What do you need? More processing power? Additional tools?"

#### Optimus (Intelligence Analyst)
**Description:** Analyzing network patterns and digital footprints.
**Dialogue Tree:**
- Traffic analysis: "Injection point detected at maintenance port. Clever entry method."
- Pattern recognition: "Timestamp metadata shows deliberate manipulation. Same skew pattern."
- Probability assessment: "Override success probability: 67%. Parameters adjusting."

#### ZTech (Technical Specialist)
**Description:** Leading the technical breach investigation and countermeasures.
**Dialogue Tree:**
- System assessment: "Air-gapped? That's what they all say until they're not."
- Vulnerability analysis: "Multiple entry points available. USB drives, maintenance ports, wireless."
- Technical solutions: "I can create a backdoor for you, or patch the existing breach."

#### Security Analyst
**Description:** IT department expert assisting with the investigation.
**Dialogue Tree:**
- System monitoring: "Deployment sequence initiated! Four minutes to deployment!"
- Evidence preservation: "We need to stop this without corrupting the evidence."
- Technical support: "I can help with authentication overrides and system commands."

## Scene 1: Server Room Investigation

### Location: Climate-Controlled Server Room
**Description:** Temperature-controlled data center with server racks, monitoring consoles, and network equipment.

**Interactive Elements:**
- **Main Server Rack** (OBSERVE): Examine primary systems
- **Network Switch** (ENGAGE): Check connection status
- **Maintenance Port** (KIT): Access service panel
- **Monitoring Console** (ENGAGE): View system logs
- **Cooling System** (OBSERVE): Environmental controls

**Puzzle: Traffic Pattern Analysis**
- **Objective:** Identify unauthorized network injection points
- **Mechanics:** Use diagnostic tool to visualize and analyze data flow
- **Solution:** Unauthorized connection through maintenance port
- **Clue:** Timestamp metadata shows manipulation patterns

**Environmental Challenges:**
- **Temperature Control:** Must maintain server room conditions
- **Security Systems:** Cameras and access logs track movement
- **Time Pressure:** System status changes require quick responses

## Scene 2: Operations Center Response

### Location: Network Operations Center
**Description:** High-tech monitoring facility with multiple screens, analyst workstations, and alert systems.

**Interactive Elements:**
- **Main Monitoring Wall** (OBSERVE): System status overview
- **Analyst Workstation** (ENGAGE): Access detailed logs
- **Security Console** (PROTOCOL): Override controls
- **Backup Systems** (KIT): Emergency access panel
- **Communication Terminal** (ENGAGE): Contact team

**Puzzle: Deployment Interruption**
- **Objective:** Stop malicious firmware deployment while preserving evidence
- **Mechanics:** Real-time countdown with multi-stage authentication
- **Solution:** Override at authentication checkpoint using backdoor credentials
- **Clue:** System cache contains hidden access credentials

**Race Against Time:**
- **4-Minute Countdown:** Deployment sequence actively running
- **Multi-Stage Process:** Authentication, validation, distribution phases
- **Evidence Preservation:** Must avoid corrupting digital fingerprints

## Scene 3: Digital Forensics

### Location: Forensic Analysis Lab
**Description:** Secure analysis environment with specialized computers, data recovery tools, and evidence storage.

**Interactive Elements:**
- **Forensic Workstation** (ENGAGE): Analyze captured data
- **Data Recovery Tools** (KIT): Extract deleted files
- **Log Analysis Software** (OBSERVE): Examine system records
- **Evidence Locker** (PROTOCOL): Secure findings
- **Network Isolation Chamber** (ENGAGE): Test malware safely

**Puzzle: Breach Method Reconstruction**
- **Objective:** Trace complete hacker entry and operation method
- **Mechanics:** Piece together digital evidence from multiple sources
- **Solution:** Compromised workstation led to server hijacking
- **Clue:** IP address correlation with physical access logs

**Final Resolution:**
- **Server Reclamation:** Remove malicious code and restore control
- **Breach Sealing:** Patch vulnerabilities and strengthen security
- **Evidence Compilation:** Document complete attack chain
