# Crimson Compass Visual Prompts Guide

## Overview
This guide provides comprehensive techniques for creating stunning visual prompts to achieve next-level graphics for the Crimson Compass iPhone mystery game. Focus on AI image generation tools like Midjourney, DALL-E, and Stable Diffusion.

## Core Principles for Effective Visual Prompts

### 1. Structured Prompt Architecture
- **Subject First**: Start with the main element (e.g., "Victorian detective character")
- **Style Specification**: Add "mobile game art style, stylized illustration, high contrast"
- **Technical Details**: Include "iPhone optimized, 1080p resolution, touch-friendly UI"
- **Atmospheric Elements**: "Eerie mystery atmosphere, dramatic lighting, immersive composition"

### 2. Mobile-Specific Optimizations
- **Resolution**: Target 1920x1080 (9:16 aspect ratio) for modern iPhones
- **Performance**: Emphasize "low-poly, optimized for mobile, efficient rendering"
- **Style**: Use "cel-shaded, flat design, vector art" for clarity and performance
- **Color**: High contrast palettes (dark blues/grays for mystery themes)

## Advanced Prompt Techniques

### Lighting & Atmosphere (Critical for Mystery Games)
- Use "dramatic chiaroscuro lighting, long shadows, volumetric fog"
- Add "eerie atmosphere, tense mood, suspenseful composition"
- Examples: "candlelit investigation scene" or "moonlit crime scene"

### Composition for Immersion
- "Rule of thirds, foreground interest, atmospheric depth"
- "Cinematic framing, character focal point, environmental storytelling"
- Mobile consideration: "vertical composition, thumb-accessible UI elements"

## Best Practices & Tools

### Prompt Refinement
- Start simple, add complexity iteratively
- Use weights: "glowing runes::1.5" to emphasize elements
- Test variations: Change lighting, colors, or perspectives
- Reference existing art: "in the style of Sherlock Holmes illustrations"

### Mobile Optimization
- Add "low-poly 3D" or "2D sprite sheet" for assets
- Specify "Unity game asset, optimized for iOS"
- Test generated images at actual iPhone resolutions

### Recommended Tools
- **Midjourney**: Best for stylized game art (`/imagine` prompts)
- **DALL-E**: Good for UI elements and icons
- **Stable Diffusion**: Free, customizable for specific styles
- **ArtStation**: Study professional game art portfolios
- **Unity Asset Store**: Reference mobile game art packs

## Implementation Tips for Crimson Compass

1. **Create Asset Folders**: Organize by type (UI, backgrounds, characters, props)
2. **Batch Generation**: Generate multiple variations of key assets
3. **Style Consistency**: Use consistent descriptors across all prompts (e.g., "noir mystery style, Victorian era")
4. **Performance Testing**: Ensure generated images compress well (under 1MB each)
5. **Iterative Process**: Generate → Review → Refine prompts → Regenerate

## Mystery Game Visual Themes
- **Atmospheric Design**: Fog, shadows, desaturated blues for tension
- **Character Integration**: Weathered clothing matching ruined environments
- **Environmental Storytelling**: Clues hidden in background details
- **Pacing Through Visuals**: Progressive reveals and interactive lighting
