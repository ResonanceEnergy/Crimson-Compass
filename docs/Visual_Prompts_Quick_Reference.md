- Add: "high contrast shadows, atmospheric lighting, cel-shaded illustration"
- End with: "mobile game art, stylized --ar [ratio] --v 6 --q 2"

### For Variations
- Change lighting: "dramatic chiaroscuro, volumetric fog, neon highlights"
- Adjust mood: "tense atmosphere, mysterious ambiance, suspenseful composition"
- Add details: "dust motes, rain-slicked surfaces, holographic interference"

### For Optimization
- Mobile-first: "iPhone optimized, touch-friendly, performance efficient"
- Style consistency: "matching previous character designs, unified color palette"
- Technical: "low-poly elements, efficient rendering, compressed textures"
