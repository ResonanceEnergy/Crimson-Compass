# Crimson Compass Visual Production - Quick Start Guide

## Immediate Next Steps (Start Today)

### Day 1: Setup & First Assets
1. **Open Midjourney** and set up your account
2. **Generate Title Logo** using this prompt:
   ```
   CRIMSON COMPASS mystery game title, ornate Victorian typography, gold embossed letters, dark parchment background, dramatic lighting, professional game logo, high contrast, mobile optimized --ar 16:9
   ```
3. **Create Main Menu Background**:
   ```
   Victorian mystery menu background, subtle fog, antique compass rose watermark, dark blue gradients, atmospheric lighting, mobile game menu, elegant and mysterious --ar 16:9
   ```

### Day 2: Core UI Buttons
1. **Play Button (3 states)**:
   ```
   Victorian play button, ornate gold frame, mystery game UI, flat design, high contrast, touch-friendly, normal state --ar 1:1
   ```
   (Generate hover/pressed by changing "normal state" to "hover state" and "pressed state")

2. **Settings & Credits buttons** using similar prompts

### Day 3: Dialogue System
1. **Dialogue Box Frame**:
   ```
   Ornate Victorian dialogue box frame, scrollwork design, gold accents, mystery game UI panel, detailed vector art, semi-transparent background ready, mobile optimized --ar 16:9
   ```

### Day 4: First Background
1. **Victorian Library**:
   ```
   Abandoned Victorian library at dusk, towering bookshelves with hidden passages, dust motes in golden hour light, mystery adventure game background, stylized illustration, atmospheric depth, muted sepia tones, high detail, mobile optimized --ar 9:16
   ```

## Daily Workflow

1. **Morning**: Review previous day's generations, select best versions
2. **Midday**: Generate new assets using prompt templates
3. **Afternoon**: Post-process selected images (cleanup, resize)
4. **Evening**: Test in Unity, adjust prompts for next day

## Quality Checks (Do Daily)

- [ ] High contrast for mobile readability
- [ ] Consistent Victorian/mystery style
- [ ] Touch-friendly button sizes
- [ ] File size under 500KB for UI, 1MB for backgrounds
- [ ] Seamless integration with existing programmatic UI

## Tools Setup

1. **Midjourney**: Join Discord, use `/imagine` commands
2. **Image Editor**: Photoshop/GIMP for post-processing
3. **Unity**: Import and test assets immediately
4. **Reference**: Keep prompt templates open for quick copying

## Success Metrics

- **Day 1-2**: 10-15 UI elements generated
- **Day 3-7**: Complete menu system functional
- **Week 1 End**: Basic game visually playable
- **Week 2 End**: All core UI professional quality

## If Stuck

1. **Simplify prompts**: Remove complex details, focus on essentials
2. **Use variations**: Add "--v 2" or "--v 3" for different styles
3. **Reference existing**: Use "in the style of [game name]" for inspiration
4. **Ask community**: Share prompts in Midjourney Discord for feedback

## Remember

- **Consistency over perfection**: Better to have consistent good assets than perfect but mismatched ones
- **Mobile first**: Always test at actual iPhone screen size
- **Iterate quickly**: Generate, review, refine, repeat
- **Document everything**: Save successful prompts for future use
