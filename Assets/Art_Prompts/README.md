# Crimson Compass Art Prompts

This folder contains AI image generation prompts for creating visual assets for the Crimson Compass mystery game.

## Folder Structure
- `UI_Elements.txt` - Prompts for buttons, icons, panels, and interface elements
- `Backgrounds.txt` - Prompts for scene backgrounds and environments
- `Characters.txt` - Prompts for character portraits and NPCs
- `Props_Items.txt` - Prompts for collectibles, evidence, and interactive objects

## Usage Instructions

1. **Choose Asset Type**: Select the appropriate .txt file based on what you're creating
2. **Copy Base Prompt**: Use the prompts as starting points
3. **Customize**: Modify descriptors to match your specific needs
4. **Generate**: Use in Midjourney, DALL-E, or Stable Diffusion
5. **Iterate**: Refine based on results and game requirements

## Key Parameters
- `--ar 9:16`: Vertical mobile aspect ratio
- `--ar 1:1`: Square icons/buttons
- `--ar 3:4`: Character portraits
- Add "mobile optimized" for performance considerations

## Style Consistency
All prompts include:
- "mystery game" or "mystery adventure"
- "mobile optimized" or "iPhone display quality"
- "high contrast" for readability
- "atmospheric" for immersion

## Tools Recommended
- Midjourney: Best for stylized illustrations
- DALL-E: Good for UI elements
- Stable Diffusion: Free and customizable

See `docs/Visual_Prompts_Guide.md` for complete documentation and best practices.
